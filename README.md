# Computer Science and Network Security Labs

This repository contains a collection of laboratory reports completed during my cybersecurity and computer science studies.

The reports focus on practical aspects of network and system security, including attack analysis, defensive mechanisms, secure system configuration, authentication, access control, and network monitoring.

## Skills Demonstrated

* Network Security
* System Hardening
* Authentication and Authorization
* Access Control Lists (ACLs)
* Traffic Analysis
* Intrusion Detection and Prevention Systems (IDS/IPS)
* Secure Remote Administration
* Security Assessment and Analysis
* Linux Security
* Cybersecurity Fundamentals
